# 扩散模型基本框架

你眼中看似落叶纷飞变化无常的世界，实际只是躺在上帝怀中一份早已谱好的乐章。

扩散模型作为生成模型，从概念上是非常简单的。

# 什么是生成模型

我们先从一个简单的场景出发，来理解生成模型。 大家在编程中都会涉及到随机数生成问题。关于随机数生成，我们一般都有如下的要求：

  • 抽取出来的数据范围是[0,1]
  • 每次抽取的数据尽可能不一样

电脑中是如何实现这个功能的？下面我们介绍一下[0,1]之间的伪随机数生成办法。可能有助于大家理解生成模型参考 (opens new window): 这篇文章重点解读一下，作为非常好的生成模型的例子。对于大家理解生成模型有帮助。

# 生成模型的物理模型

从本质看，我们获得一批样本，如果能够直接通过此样本求解出其对应的概率分布，那么，最简单的办法就可以采取蒙特卡洛抽样法，从这个概率分布中抽样，实现生成新样本。

可惜的是，一般我们无法求解出样本本身对应的概率分布。即使我们知道了样本的概率分布，这个分布可能会非常复杂，如何从这样的分布中高效采样也是个难题。 那么，研究人员就希望把这个问题进行转换： P(X)===>Q(Y) 其中Q是相对简单的，一般都采取的是正态分布。 这样，整个问题就转化为如何将原始的数据分布转化成标准的概率分布，然后在将此正态分布生成原始的分布问题。

P(X)===>Q(Y)====>P(X)

2015年，作者基于其物理学背景，借鉴物理学中的掺杂与去杂过程，引入到生成领域。 这样，在技术上，我们逐步向一幅图中添加噪声，逐步叠加后，最终此图像就变成完全随机的一幅图。然后，从完全随机的一幅图中，按照某种策略，逐步去除噪声，就可以还原出图像。

这种基于物理思维的实验驱动的模式，也引发AI领域新的研究模式：

  • 通过建立物理模型，构建可解释的过程
  • 在此基础上引入数学物理框架，形成新型算法 这也是为什么扩散模型获得如此众多研究者的喜好原因之一。
