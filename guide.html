# 指南

本文提供如何使用GenMo产品的操作指南。 GenMo是一款桌面产品，提供基于文本生成动作的AI工具。

# 下载与安装

# 下载

您可以通过这里下载GenMo安装包。

# 安装

下载安装包到本地电脑上，点击install.exe即可进行本地安装。 安装成功后，选择genmo.exe执行启动程序。

# 初始化

genmo程序启动成功后，会提示进行环境初始化。主要包括两个部分：

  • 下载需要的模型数据。一旦模型数据下载到本地缺省目录中，请不要修改此目录
  • 运行环境初始化，主要是检测本地的GPU卡，并根据GPU卡的型号安装相关的依赖包。

注意：由于模型数据及GPU依赖包比较大，此过程比较长，需要耐心等待。
